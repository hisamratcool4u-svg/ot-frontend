# OT Frontend Pro (Vite + React + Tailwind + FullCalendar)

## Quick start
1. unzip
2. `npm install`
3. set environment variable `VITE_API_BASE` if you want different backend (default uses https://ot-backend-g90m.onrender.com/api)
4. `npm run dev` (development)
5. `npm run build` (production)

## Vercel
This project is Vite-based and deploys on Vercel. Ensure build command is `npm run build` and output directory is `dist/`.
